# Guía para Staff

**Última actualización:** 2026-04-22
**Estado:** Borrador — módulos en desarrollo

---

## Acceso

El dueño o manager te invita por email. Recibes un link para crear tu contraseña y acceder al panel.

## Vista de agenda personal

- Ves solo tus citas asignadas (no las de otros profesionales)
- Vista diaria y semanal
- Colores por tipo de servicio

## Gestión de citas

Para cada cita puedes:
- Ver detalles: cliente, servicio, hora, notas
- Marcar como `en progreso` cuando el cliente llega
- Marcar como `completada` al terminar
- Registrar `no-show` si el cliente no aparece
- Agregar notas internas (no visibles para el cliente)

## Lo que NO puedes hacer como Staff

- Ver citas de otros profesionales
- Modificar precios o servicios
- Acceder a información financiera
- Cancelar citas de otros
